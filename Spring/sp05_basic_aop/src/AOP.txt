1.AOP(관점지향 프로그래밍)
*관심사
 1. 주관심
 2. 보조관심
 
 ex) 계산기
     ==>주업무   :(사칙연산) + , - , * , / 
     ==>보조업무 :계산에 걸린 시간을 구하고 싶다
     
 AOP > 주관심와 보조관심사 혼재 되어있는 코드
         사칙연산 진행 => 보조관심사 => Error
         개발자 입장에서 주관심사 코딩 => 보조관심사 같이(x)
         
 AOP : 일반 자바 구현
       Spring 좀 더 다양한 구현 (코드 편리성 제공)
 
ex)  웹 프로그래밍 (회원관리)
      주관심사 : 회원정보 => DB 연결 => Insert
      보조관심사 : 한글처리(encoding)
                    로그처리 (접속,실행 흐름)
                    트랜잭션 처리 등등 ...
